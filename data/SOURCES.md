# OVERVIEW

This document is a synopsis of the datasets contained within this project.
The **data** directory is compartmentalized into **raw** and **processed** subdirectories.

## About

Within the raw subdirectory, all files are stored as-is when obtained from their source.

Any data resulting from preprocessing, exploration, or computing is saved in the processed subdirectory.

Each file shall be documented below, with information detailing the source, method obtained, description, and any applicable notes.
